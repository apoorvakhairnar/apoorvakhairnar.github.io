**The Goal:** Develop an inventory management system for grocery stores using Machine Learning and Mechanical Engineering principles.

**My Role/The Solution:** I co-developed a Convolutional Neural Network (CNN) to identify grocery items using a smartphone camera, paired with a custom GUI and SQL database for inventory tracking.

**Tools:** Python, Convolutional Neural Networks (CNN), SQL, Object Detection
