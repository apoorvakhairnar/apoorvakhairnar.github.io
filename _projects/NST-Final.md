**The Goal:** Review existing Lyapunov-based nonlinear control techniques for soft robots, focusing on Sliding Mode Control (SMC).

**My Role/The Solution:** Surveyed various implementations of SMC for soft robots using models like Piecewise Constant Curvature and Cosserat Rods. Compared merits, drawbacks, and key features of these algorithms.

**Tools:** Sliding Mode Control (SMC), Nonlinear Control Theory
